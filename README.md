Библиотека элементов для Altium

### Содержание

* *rules* - настройки правил, на данный момент есть только для резонита
* *STEP* - 3Д модели элементов
* *Footprints* - Посадочные места для элементов
* *Symbols* - Условные обозначения элементов

### Сенсоры

* *BMP280*
* *BME280*
* *MPU9250*
